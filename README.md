# Advanced Radio System (ESX/QBCore)

**The ultimate communication solution for FiveM servers.** This script replaces the standard radio with a sleek, interactive, and fully customizable handheld radio interface. Powered by `pma-voice`, it features a drag-and-drop UI, job-restricted channels, frequency history, and immersive sound effects.
